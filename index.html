Understanding the Code

Staff Type (1st Letter)

D = Diplomat, C = Consular, S = Staff, A = Admin & Technical, M = Mission

Country Code (2nd & 3rd Letters)

Two-letter code intentionally obscure for security. For example: AF = Japan, FP = UK

Format

Example: "DAF" = Diplomat (D) from Japan (AF). Codes are assigned by the US State Department.
